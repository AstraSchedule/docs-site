# 管理端相关

## 管理端显示白屏？

1. 打开浏览器开发者工具（F12），查看 Console 和 Network 标签
2. 如果请求后端 API 失败，检查：
   - 后端是否运行
   - `src/global.js` 中 `APISRV` 地址是否正确（必须与部署时配置一致）
3. 如果是 Cloudflare Pages 部署，确认构建成功且 `APISRV` 地址包含了协议（`https://`）
4. 尝试清除浏览器缓存（Ctrl+Shift+Delete）

## 保存配置失败？

1. 返回 401：认证失败。检查 JWT Token 是否过期，或密码确认是否正确。
2. 返回 403：WAF 拦截。按部署相关章节的方式排查 Cloudflare WAF 规则。
3. 返回 500：后端内部错误。查看后端日志定位具体原因（可能是数据库写入失败或数据格式不对）。
4. 请求超时：函数计算冷启动或其他网络问题，稍后重试。
5. 网络错误（Failed to fetch）：后端未运行或网络不通。

## 管理端登录密码忘记了怎么办？

管理端使用 JWT 登录认证。

1. 访问管理端登录页面，输入用户名和密码
2. 默认管理员账号：`admin` / `admin`（首次登录需修改密码）

密码存储在服务器 `config.toml` 中：

```toml
[secret]
token = "你的密码"
```

如果可以访问服务器，查看该文件即可获取密码。密码被泄露需要更改的话，直接修改这个值并重启后端。

用户名可通过管理端用户管理页面修改（admin 角色）。
