# 调试选项

本文档介绍托盘菜单中的调试选项子菜单，包含调试输入和调试矫正两个功能。

## 调试选项子菜单

| 属性 | 值 |
|------|-----|
| 类型 | 子菜单 |
| 位置 | 托盘菜单 → 调试选项 |

点击后展开子菜单，包含以下两个选项：

### 调试输入

| 属性 | 值 |
|------|-----|
| 类型 | 输入框 |
| 默认值 | `0` |
| 存储键 | `debugInputValue` |

输入调试值，当前用于气温偏移校准。修改后立即生效并持久化保存。

**使用场景**：当客户端显示的温度与实际体感存在偏差时，可通过此功能进行偏移校准。正数表示升高显示温度，负数表示降低。

### 调试矫正

| 属性 | 值 |
|------|-----|
| 类型 | 按钮 |

快速将课表计时矫正到指定时间点。点击后弹出三步对话框：

1. **选择课节**：从当前时间表中选择要矫正到的课节
2. **选择时机**：选择"上课前"或"下课后"
3. **输入秒数**：输入偏移的秒数（默认 5 秒）

系统会自动计算时间偏移量并应用。例如选择"第 3 节课上课前 5 秒"，系统会将时间调整到第 3 节课开始前 5 秒的时刻。

**注意**：timetable 中的结束时间比实际结束时间少 1 分钟（如配置 `08:00-08:39` 实际课程到 `08:40`），选择"下课后"时会自动加 60 秒。
