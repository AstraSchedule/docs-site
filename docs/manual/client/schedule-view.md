# 课表与视图

本文档介绍托盘菜单中与课表显示和视图控制相关的菜单项。

## 当前地区

| 属性 | 值 |
|------|-----|
| 类型 | 输入框 |
| 默认值 | 空字符串 |
| 存储键 | `local` |

设置天气查询的地理位置（城市名称），如"北京""上海"。用于和风天气 API 查询。

## 更新课表

| 属性 | 值 |
|------|-----|
| 类型 | 按钮 |

立即从服务端拉取最新配置。Serverless 模式直接请求，其他模式通过 WebSocket 广播同步指令。

## 配置课表

| 属性 | 值 |
|------|-----|
| 类型 | 按钮 |

临时修改当前某节课的科目，仅本次运行有效，重启后恢复。点击后弹出两步对话框：

1. 选择要修改的课程时段
2. 从所有可用科目中选择替换科目

此修改仅在当前运行实例的内存中生效，不会持久化或同步到服务端。

## 矫正计时

| 属性 | 值 |
|------|-----|
| 类型 | 输入框 |
| 默认值 | `0` |
| 存储键 | `timeOffset`（localStorage） |

设置课表计时与系统时间的偏移秒数。正数为延后，负数为提前。用于校正客户端显示时间与实际时间的偏差。

## 切换日程

| 属性 | 值 |
|------|-----|
| 类型 | 子菜单 |

强制使用本周某一天的课表替代当天课表，自动在日期变化后重置。菜单中提供"重置至当前日期"选项以立即恢复。

适用场景：临时调课、考试安排等需要显示非当天课表的情况。
