# 显示与开关

本文档介绍托盘菜单中与显示模式和系统行为相关的开关选项。

## 课上计时

| 属性 | 值 |
|------|-----|
| 类型 | 复选框 |
| 默认值 | 开启 |
| 存储键 | `isDuringClassCountdown` |

上课时段是否显示倒计时。开启后，上课时段会在迷你倒计时窗口中显示当前课程的剩余时间。

## 窗口置顶

| 属性 | 值 |
|------|-----|
| 类型 | 复选框 |
| 默认值 | 开启 |
| 存储键 | `isWindowAlwaysOnTop` |

窗口是否保持在所有窗口的最顶层。开启后使用 `screen-saver` 级别的置顶策略，确保课表窗口不会被其他窗口遮挡。

## 始终缩小

| 属性 | 值 |
|------|-----|
| 类型 | 复选框 |
| 默认值 | 关闭 |
| 存储键 | `isAlwaysMinimized` |

是否始终以迷你倒计时窗口显示。开启后，课表窗口将始终显示为精简的迷你倒计时模式，不显示完整课表。

## 上课隐藏

| 属性 | 值 |
|------|-----|
| 类型 | 复选框 |
| 默认值 | 开启 |
| 存储键 | `isDuringClassHidden` |

上课时段是否隐藏课表主体和普通倒计时框。开启后，上课时段仅保留迷你倒计时窗口，减少对学生注意力的干扰。

## 开机启动

| 属性 | 值 |
|------|-----|
| 类型 | 复选框 |
| 默认值 | 开启 |
| 存储键 | `isAutoLaunch` |

是否随 Windows 系统自动启动。开启后，客户端会在用户登录时自动启动。

## 切换图标

| 属性 | 值 |
|------|-----|
| 类型 | 按钮 |
| 存储键 | `trayIcon` |

切换系统托盘图标的样式。点击后在默认图标和应用图标之间切换，立即生效。

## 显示模式组合

通过"课上计时""上课隐藏""始终缩小"三个开关，可以组合出四种不同的显示模式：

| 课上计时 | 上课隐藏 | 始终缩小 | 效果 |
|----------|----------|----------|------|
| 开启 | 关闭 | 关闭 | 完整显示课表和倒计时（推荐） |
| 开启 | 开启 | 关闭 | 上课时段仅显示迷你倒计时 |
| 开启 | 关闭 | 开启 | 始终显示迷你倒计时 |
| 关闭 | - | - | 不显示倒计时 |
