# 云端与更新

本文档介绍托盘菜单中与云端连接和自动更新相关的菜单项。

## 连接云端

| 属性 | 值 |
|------|-----|
| 类型 | 复选框 |
| 默认值 | 关闭 |
| 存储键 | `isFromCloud` |

标记是否从云端获取课表。当前为预留功能，无实际操作。

## 更新源（可选）

| 属性 | 值 |
|------|-----|
| 类型 | 输入框 |
| 默认值 | 空（使用默认 GitHub Releases 镜像） |
| 存储键 | `updateBaseUrl` |

设置自定义自动更新镜像源地址，需指向包含 `latest.yml` 的目录。修改后需重启应用生效。

## 检查更新

| 属性 | 值 |
|------|-----|
| 类型 | 按钮 |

立即手动检查是否有新版本可用。仅在打包版本（`app.isPackaged`）中生效，开发模式下会提示"开发模式下不检查更新"。

## 自动更新

| 属性 | 值 |
|------|-----|
| 类型 | 复选框 |
| 默认值 | 开启 |
| 存储键 | `isAutoUpdate` |

是否启用自动更新检测。开启后，应用启动 3 秒后自动检查更新。

## 云端服务

| 属性 | 值 |
|------|-----|
| 类型 | 输入框 |
| 默认值 | `class.khbit.cn` |
| 存储键 | `server` |

设置后端 API 服务器地址。修改后触发重新拉取配置。

## 安全连接

| 属性 | 值 |
|------|-----|
| 类型 | 复选框 |
| 默认值 | 开启 |
| 存储键 | `isSecureConnection` |

是否使用 HTTPS/WSS 加密连接。关闭后使用 HTTP/WS。

## 所在班级

| 属性 | 值 |
|------|-----|
| 类型 | 输入框 |
| 默认值 | `39/2023/1` |
| 存储键 | `class` |

设置班级标识，格式为 `学校/年级/班级`（如 `39/2023/1`）。修改后触发 WebSocket 重连。

## 刷新天气

| 属性 | 值 |
|------|-----|
| 类型 | 按钮 |

立即手动刷新天气数据。
