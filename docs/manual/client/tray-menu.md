# 托盘菜单总览

右键点击系统托盘中的星程课表图标（或左键点击图标、或点击课表窗口中的星期标签），弹出托盘菜单。菜单包含以下功能项（按顺序排列）：

## 完整菜单结构

```
├── 连接云端 ☑
├── 更新源(可选)
├── 检查更新
├── 自动更新 ☑
├── 云端服务
├── 安全连接 ☑
├── 所在班级
├── 刷新天气
├── 当前地区
├── 更新课表
├── ─────────────
├── 配置课表
├── 矫正计时
├── 切换日程
├── ─────────────
├── 调试选项 ▸
│   ├── 调试输入
│   └── 调试矫正
├── 源码仓库
├── ─────────────
├── 课上计时 ☑
├── 窗口置顶 ☑
├── 始终缩小 ☐
├── 上课隐藏 ☑
├── 开机启动 ☑
├── 切换图标
├── ─────────────
└── 退出程序
```

## 菜单分类

| 分类 | 包含项 | 详细页面 |
|------|--------|----------|
| 云端与更新 | 连接云端、更新源、检查更新、自动更新、云端服务、安全连接、所在班级、刷新天气 | [云端与更新](./cloud-update) |
| 课表与视图 | 当前地区、更新课表、配置课表、矫正计时、切换日程 | [课表与视图](./schedule-view) |
| 调试选项 | 调试输入、调试矫正 | [调试选项](./debug) |
| 显示与开关 | 课上计时、窗口置顶、始终缩小、上课隐藏、开机启动、切换图标 | [显示与开关](./display-switches) |
| 其他 | 源码仓库、退出程序 | - |

## 托盘提示文字

托盘提示文字（tooltip）显示当前版本号和连接状态（在线、离线、Serverless 模式），更新下载过程中会显示下载进度。
