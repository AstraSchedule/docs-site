# 客户端使用

AstraSchedule 客户端是基于 Electron 的 Windows 桌面应用程序，设计用于教室电子白板或教学一体机上长期运行。本章介绍客户端的安装、配置和日常使用方法。

## 安装与首次启动

从 GitHub Releases 页面下载最新版本的安装包（`AstraScheduleInstaller.exe`），运行安装程序完成安装。安装完成后，客户端会自动启动并显示课表窗口。

首次启动时，客户端使用内置的默认配置和默认服务端地址（`class.khbit.cn`）。如果使用自建服务端，需要通过托盘菜单修改服务端地址和班级标识。服务端地址和班级标识配置正确后，客户端将自动从服务端拉取课表、作息、天气等数据。

客户端默认为开机自启动，无需手动添加到启动文件夹。此行为可在托盘菜单中关闭。

## 功能概览

客户端提供以下核心功能：

- **课表显示**：实时显示当前课程、倒计时、星期信息
- **天气联动**：显示实时气温和天气状况，支持温度颜色映射
- **倒数日**：独立窗口显示距离考试、放假等目标日期的天数
- **横幅通知**：顶部跑马灯滚动通知，支持天气预警替代
- **集控管理**：通过云端服务端统一配置，客户端自动同步
- **窗口特性**：支持置顶、点击穿透、透明度调节
