# 配置持久化

用户通过托盘菜单修改的设置会自动保存，重装或更新后保持不变。

## 可持久化的设置

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 服务器地址 | `class.khbit.cn` | 后端 API 服务器地址 |
| 班级标识 | `39/2023/1` | 班级标识，格式为 `学校/年级/班级` |
| 安全连接 | 开启 | 是否使用 HTTPS/WSS 加密连接 |
| 窗口置顶 | 开启 | 窗口是否保持在最顶层 |
| 开机启动 | 开启 | 是否随系统自动启动 |
| 自动更新 | 开启 | 是否启用自动更新检测 |
| 更新源 | 空（使用默认） | 自定义自动更新镜像源地址 |
| 天气地区 | 空 | 天气查询的地理位置 |
| 课上计时 | 开启 | 上课时段是否显示倒计时 |
| 上课隐藏 | 开启 | 上课时段是否隐藏课表主体 |
| 始终缩小 | 关闭 | 是否始终以迷你倒计时显示 |
| 调试输入 | `0` | 温度偏移校准值 |
| 托盘图标 | 默认 | 托盘图标样式 |

## 注意事项

- 配置文件保存在 `%APPDATA%/electron-class-schedule/config.json`
- 卸载程序时可选择是否保留配置
- 如需重置所有设置，可手动删除配置文件后重启客户端
