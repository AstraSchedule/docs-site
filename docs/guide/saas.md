# SaaS 免部署使用

不想自己折腾服务器？SaaS 模式帮你省去部署环节，拿到域名就能用。

## 怎么运作的

我们帮您部署好了所有服务，您只需要注册一个账户，然后登录并使用即可。

## SaaS vs 独立部署

| | SaaS | 独立部署 |
|---|---|---|
| 服务器 | 我们来 | 你自己来 |
| 域名 | 用 getastra.cn 的子域名 | 自己的域名 |
| 数据库 | 我们来 | 自己使用 SQLite 或 MySQL |
| 维护 | **不用操心** | **全靠自己** |
| 费用 | **现在不收费** | 服务器 + 运维成本 |
| 定制 | 有限 | **完全自由** |

简而言之：独立部署你说了算，但得自己干；SaaS 省心，而且目前也不收费。两边功能一样，性能也没有区别。

## 开始注册新租户

浏览器访问 [go.getastra.cn](https://go.getastra.cn)

### 1. 确定一个域名

![image-20260704030036050](https://image-hk-1.oss-accelerate.aliyuncs.com/image-20260704030036050.png)

跟随提示要求，确定一个子域名。等待 **数秒** 服务端检查域名是否可用，如果不可用则很大概率是已被占据，需要您重新起一个名字。固然，为了方便他人，方便自己，尽量选择比较难重名的缩写。为了保证数据安全与防止误操作，租户 **没有办法** 删除自己，只能从系统内部进行删除，如需删除租户，请您联系 kuohu@getastra.cn

### 2. 设置管理员账户

![image-20260704030342761](https://image-hk-1.oss-accelerate.aliyuncs.com/image-20260704030342761.png)

管理员账户拥有本租户的 **最高权限** ，建议您设置 **不容易遗忘** 且 **尽量复杂** 的密码，当然您也可以把密码记在纸上，并妥善保管纸条。本租户内其他用户若遗忘密码，只能由管理员进行修改；而管理员若遗忘密码，则只能从系统内部进行修改。您需要联系 kuohu@getastra.cn

### 3. 创建空白班级

![image-20260704030859751](https://image-hk-1.oss-accelerate.aliyuncs.com/image-20260704030859751.png)

如图所示，您需要创建一个空白班级，我们建议您 **不要使用中文** ，因为这个字段也需要您手动输入到教室机器上。这个字段与子域名不同，您可以填写任意信息，这不会发生冲突或重复的问题，因为数据是隔离的。所以越简单越好。

提示给了三类，简而言之，我们建议：

|                                  | 个人用户  | 学校用户 |
| -------------------------------- | --------- | -------- |
| 学校名称格式类似“南京市第39中学” | 39/2023/1 | 39/7/1   |
| 学校名称格式类似“中华中学”       | zh/2026/1 | zh/10/1  |
| 学校使用 abcd 而不是 1234 分班   | xx/2026/a | xx/10/a  |

这是出于管理方便给出的建议，这样就免去了每年增删班级/年级的问题，当然您也可以自己定，这只是给出一个思路。

### 4. 最终确认

![image-20260704031829903](https://image-hk-1.oss-accelerate.aliyuncs.com/image-20260704031829903.png)

最终确认信息，确认无误并等待人机验证完成后点击“确认注册”即可。

![image-20260704031923588](https://image-hk-1.oss-accelerate.aliyuncs.com/image-20260704031923588.png)

点击后会显示一个进度条，初始化的过程可能较慢，同样需要等待 **数秒** 。

### 5. 注册完成！

![image-20260704032032187](https://image-hk-1.oss-accelerate.aliyuncs.com/image-20260704032032187.png)

等待出现注册成功的页面后，跟随提示操作即可。

## 登录管理后台

浏览器打开 [i.getastra.cn](https://i.getastra.cn)，用分配的账号登录。

在这里配课表、作息、倒计时，所有班级的设置都在这一个地方搞定。

如果提示您用户名或密码错误，除了用户名或密码错误外也有可能是因为租户不存在。您需要检查后端地址是否与注册成功页的地址相符。若相符则有可能是 DNS 还暂未生效，您可以等待 **数秒** 至 **数分钟** 后再试，这取决于您当地的运营商。
