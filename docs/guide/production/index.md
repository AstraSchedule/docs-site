# 外网高并发部署

> 🚀 适合多学校共用、或者追求企业级可用性的场景。

## 架构

<!-- TODO: 架构图 -->

```
客户端/浏览器 → 阿里云 ESA (DDoS + WAF + CDN) → 阿里云函数计算 (后端)
                                                → 管理端 (CDN)
                         ↓
                  阿里云 MySQL Serverless (数据库)
```

## 方案说明

| 组件 | 选型 | 说明 |
|------|------|------|
| 后端 | 阿里云函数计算 | Serverless 弹性伸缩，按量计费 |
| 数据库 | 阿里云 MySQL Serverless | 弹性伸缩，自动备份，高可用 |
| 管理端 | 静态托管 + CDN | 构建后部署到 CDN 或 Netlify |
| 安全防护 | 阿里云 ESA | 全站加速 + WAF + DDoS 防护 + HTTPS |

## 对比极低成本方案

| 对比项 | 极低成本 | 高并发 |
|--------|----------|--------|
| 数据库 | SQLite（单实例，无备份） | MySQL Serverless（自动备份，弹性伸缩） |
| 安全 | Cloudflare 免费 WAF | ESA 企业级 WAF + DDoS |
| 可用性 | 普通 | 99.95%+ |
| 成本 | ≈ 0 | 按量计费，随规模增长 |
