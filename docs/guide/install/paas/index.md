# PaaS 部署（推荐）

使用阿里云函数计算运行后端，Netlify 托管管理端，Cloudflare CDN 提供 WAF 防护。

## 架构图

<!-- TODO: 插入架构图 -->

整体流程：管理员浏览器 → Cloudflare CDN → Netlify（管理端）；客户端 → Cloudflare CDN → 函数计算（后端 API）。

## 你需要准备

- 一个域名（几十块一年，.top .xyz 等很便宜）
- 阿里云账号（注册实名即可）
- GitHub 账号（用于 Netlify 部署）
- Cloudflare 账号（免费）
- 约半小时时间

## 成本

| 项目 | 费用 |
|------|------|
| 域名 | ≈ 30 元/年 |
| 函数计算 | ≈ 0 元（免费额度内） |
| Netlify | 免费 |
| Cloudflare CDN | 免费 |
| **合计** | **≈ 30 元/年** |
