# 管理端配置

本文描述管理端配置文件各项内容的含义

## 配置文件位置

管理端配置文件位于 `src` 目录下，名为 `global.js`，内容如下：

```javascript 
export const APISRV = "https://class.khbit.cn"
```

## APISRV
APISRV 是服务端的地址，管理端会通过这个地址与服务端进行通信。请将其修改为您实际部署的服务端地址，例如 `https://api.example.com`，确保管理端能够正确访问服务端接口。

> [!DANGER]
> 这个地址应该是从外网访问服务端的地址。如果您全部在同一内网部署，则应该是内网地址。
>
> **请不要填写 `127.0.0.1` 或者 `0.0.0.0`，他们是无效的**
