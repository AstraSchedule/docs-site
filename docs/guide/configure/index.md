# 配置说明

系统有三个组件各自有配置文件，安装完毕后需要按需修改。

## 配置文件一览

| 组件 | 配置文件 | 说明 |
|------|----------|------|
| 后端 | `config.toml` | 数据库、API 密钥、端口、CORS 等核心配置 |
| 管理端 | `src/global.js` | 后端 API 地址 |
| 客户端 | `scheduleConfig.js` | 课表数据、样式、本地离线配置 |

## 配置顺序

1. 先配后端（`config.toml`），确保后端能正常启动
2. 再配管理端（`global.js`），指向后端地址
3. 最后在管理端页面中配置课表数据，或直接修改客户端本地配置
