# 专有名词释义速查表

本页面收录 AstraSchedule 文档中出现的专有名词，按字母顺序排列，每个名词提供通俗解释和专业解释。

- [API](./api) — 应用程序编程接口
- [Backend](./backend) — 后端（服务器端程序）
- [BasicAuth](./basicauth) — HTTP 基本认证
- [Best Practices](./best-practices) — 最佳实践
- [bcrypt](./bcrypt) — 密码哈希算法
- [CDN](./cdn) — 内容分发网络
- [Cloudflare](./cloudflare) — 网络安全与加速服务商
- [CNAME](./cname) — 别名记录
- [CORS](./cors) — 跨域资源共享
- [CSS 变量](./css-variables) — CSS 自定义属性
- [DDoS](./ddos) — 分布式拒绝服务攻击
- [DNS](./dns) — 域名系统
- [Frontend](./frontend) — 前端（用户界面）
- [HTTPS](./https) — 加密的 HTTP
- [JSON](./json) — JavaScript 对象表示法
- [JWT](./jwt) — JSON Web Token
- [MySQL](./mysql) — 关系型数据库
- [Namespace](./namespace) — 命名空间（多租户隔离）
- [RBAC](./rbac) — 基于角色的访问控制
- [SaaS](./saas) — 软件即服务
- [Serverless](./serverless) — 无服务器架构
- [SQLite](./sqlite) — 嵌入式数据库
- [Tenant](./tenant) — 租户（多租户架构中的独立用户群体）
- [TOML](./toml) — 配置文件格式
- [Upsert](./upsert) — 更新或插入操作
- [WAF](./waf) — Web 应用防火墙
- [WebSocket](./websocket) — 全双工通信协议
