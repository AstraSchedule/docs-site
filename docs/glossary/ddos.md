# DDoS（分布式拒绝服务攻击）

## 通俗解释

DDoS 攻击就是用大量假请求淹没你的网站，让它忙不过来，正常用户访问不了。就像一群人堵在店门口，真正的顾客进不去。

## 专业解释

DDoS（Distributed Denial of Service）是一种网络攻击，通过控制大量主机向目标发送请求，耗尽目标资源。

常见类型：
- **流量型**：发送大量数据包，耗尽带宽
- **协议型**：利用协议漏洞，消耗服务器资源
- **应用层**：模拟正常请求，消耗应用资源

防护方式：
- **流量清洗**：识别并过滤恶意流量
- **CDN 缓存**：将流量分散到边缘节点
- **速率限制**：限制单 IP 请求频率
- **黑白名单**：阻止恶意 IP
